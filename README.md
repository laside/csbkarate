# CSB Karaté - Plateforme Web

[![Déploiement Vercel](https://img.shields.io/badge/Déploiement-Vercel-black?logo=vercel)](#)
[![Base de données](https://img.shields.io/badge/Backend-Supabase-3ECF8E?logo=supabase)](#)

Site web officiel du club CSB Karaté et interface d'administration intégrée. Ce projet est une Single Page Application (SPA) optimisée pour les performances et la facilité de maintenance.

## 🛠 Stack Technique

- **Front-End :** HTML5, Vanilla JS (ES6), CSS3
- **Styling :** Tailwind CSS (via CDN)
- **Backend & Auth :** Supabase (PostgreSQL)
- **Hébergement :** Vercel
- **Architecture :** Design System modulaire ("Architecture Bento")

## ⚙️ Fonctionnalités Principales

- Interface publique : Actualités, Galerie, Grades, Boutique.
- Espace Administration sécurisé (Supabase Auth).
- Gestion dynamique du contenu sans rechargement de page.
